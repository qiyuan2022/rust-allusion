## 左侧面板（导航区）

采用 **可折叠/可调整宽度** 的三段式结构：

### 1. Locations（位置）
- **树形文件夹浏览**
- 显示已导入的文件夹层级
- 支持新增文件夹（新增后进行图片扫描）、删除文件夹
- 支持展开/折叠文件夹

**功能**：快速跳转到不同文件夹，支持多文件夹监控

---

### 2. Tags（标签）
- **标签管理**
- 显示所有可用标签
- 支持标签层级（父子标签）

**功能**：
- 点击标签筛选图片
- 编辑或删除标签

---

---

## 顶部工具栏

从左到右：

| 图标 | 功能 | 说明 |
|------|------|------|
| `←` | 返回 | 浏览历史后退 |
|搜索框|占据主要位置，支持文本搜索和清空|
| `□ 0/46` | 选择计数器 | 当前选中 0 张 |
| `≡` | 列表/网格切换 | 切换布局模式 |
| `⬆` | 排序 | 按日期/名称/大小等排序 |

---

## 主内容区（画廊）

### 图片列表预览
- 支持**瀑布流/网格布局**
- 每张缩略图包含：
  - 图片，图片底部显示其标签
  - 选中图片，支持多选，选中后右键支持打标签

---

## 核心交互模式

### 1. 浏览流程
```
文件夹树选择 → 网格浏览 → 点击放大预览 → 打标签/评分 → 下一张
```

### 2. 筛选流程
```
点击标签 ←→ 使用搜索 ←→ 保存搜索条件 ←→ 快速调用
```

### 3. 组织流程
```
多选图片（Ctrl/Shift）→ 批量打标签 → 添加到收藏 → 导出/移动
```

## 设计特点总结

| 特点 | 说明 |
|------|------|
| **三栏式布局** | 导航-内容-详情（截图未显示右侧详情栏）|
| **扁平化设计** | 无阴影，简洁边框，白色背景 |
| **信息密度适中** | 缩略图大小可调，适应不同屏幕 |
| **视觉优先** | 图片占主要区域，UI 元素精简 |
| **渐进式功能** | 基础功能显眼，高级功能折叠在菜单中 |
